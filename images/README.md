# Images Folder - Ben J Stevens Ministry Website

## Required Images (Replace Placeholders)

Place your images in this folder with the following names. All images should be optimized for web (recommended: JPEG for photos, max 1920px wide, compressed to ~100-200KB).

### Homepage
- `hero-bg.jpg` - Hero section background (sunlight/spiritual theme, 1920x1080 recommended)
- `profile-placeholder.jpg` - Profile photo for bio section (800x1000 recommended, 4:5 ratio)
- `speaking-bg.jpg` - Speaking CTA section background (preaching/stage photo, 1920x800)

### Sermons Page
- `sermon-1.jpg` through `sermon-6.jpg` - Sermon thumbnails (800x450, 16:9 ratio)
- `series-1.jpg`, `series-2.jpg`, `series-3.jpg` - Series cover images (600x800, 3:4 ratio)

### Bookings Page
- `booking-bg.jpg` - Bookings hero background (ministry/speaking photo, 1920x1080)
- `worship-placeholder.jpg` - What to Expect section image (800x600, 4:3 ratio)

### About Page
- `gallery-1.jpg` through `gallery-6.jpg` - Ministry gallery photos
  - `gallery-1.jpg` and `gallery-6.jpg` should be wide (800x450, 16:9 ratio)
  - `gallery-2.jpg` through `gallery-5.jpg` should be square (600x600, 1:1 ratio)

### Blog Page
- `blog-featured.jpg` - Featured blog post image (800x600, 4:3 ratio)
- `blog-1.jpg` through `blog-5.jpg` - Blog post thumbnails (800x500, 16:10 ratio)

## Image Guidelines

1. **Quality**: Use high-resolution photos that are sharp and well-lit
2. **Compression**: Optimize all images for web to improve load times
3. **Style**: Maintain warm, professional aesthetic throughout
4. **People**: Ensure you have permission to use any photos with people
5. **Consistency**: Use similar color grading/filters across all images
